# Conway's Game of Life

## A playable version of Conway's Game of Life. 

![Screenshot of Game](/screenshot.gif)

## Instructions

Move your red dot around and eat the blue dots to grow in size.

## Controls

| Action | Key |
| --- | --- |
| Move | Arrow keys or ijkl |
| Reset | Enter |
| Pause | P |
| Increase Size | Space | 
| Spawn something | y |
| Debug info | d |
| Draw graph | g |
